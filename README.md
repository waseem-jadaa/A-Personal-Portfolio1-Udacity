# Personal Portfolio

A responsive personal portfolio website showcasing my projects, skills, and background. Built with HTML, CSS, and JavaScript for a clean and modern user experience.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices.
- **Dynamic Content**: Loads project and personal information from JSON data files.
- **Interactive Elements**: Smooth scrolling, hover effects, and interactive project showcases.
- **Clean UI**: Minimalist design with a focus on readability and user engagement.

## Technologies Used

- **HTML5**: Semantic markup for structure.
- **CSS3**: Custom styling with normalize.css for consistency.
- **JavaScript**: Dynamic content loading and interactivity.
- **JSON**: Data storage for projects and about me sections.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/udacity/nd0011-cd0428-starter
   ```
2. Navigate to the project directory:
   ```bash
   cd personal-portfolio
   ```

3. Open `index.html` in your web browser to view the portfolio.

No additional dependencies or build steps are required as this is a static website.

## Usage

- Open `index.html` in any modern web browser.
- Navigate through the sections to view about me information and projects.
- The website is fully client-side and does not require a server.
